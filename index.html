What is Hosana?

Hosana is OpenXeroth’s first open-source project: a real-time wildlife re-identification pipeline. Built for 24/7 video streams, it leverages state-of-the-art detection and transformer-based re-ID to detect, track, classify and identify individual animals.

Pipeline Architecture

Hosana processes live video in real time using YOLOv8 detection, object tracking, species classification, and Vision Transformer-based re-identification, with a feedback loop via Slack.

Join the Mission

OpenXeroth is community-powered. Whether you're an engineer, conservationist, or volunteer, you can contribute to Hosana. Improve the pipeline, validate re-IDs, or help design integrations with platforms like Wildbook.
