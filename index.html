What is Hosana?

Hosana is OpenXerothâ€™s first open-source project: a real-time wildlife re-identification pipeline. Built for 24/7 video streams and drone feeds, it leverages state-of-the-art detection and transformer-based re-ID to track individual animals and trigger conservation action.

Pipeline Architecture

Hosana processes live video in real time using YOLOv8 detection, object tracking, species classification, and Vision Transformer-based re-identification, with a feedback loop via Slack.


graph LR
    cam[Camera Stream] --> detect[YOLOv8 Detector]
    detect --> track[Tracking]
    track --> classify[Species Classifier]
    classify --> reid[Transformer Re-ID]
    reid --> feedback[Slack Feedback]

Join the Mission

OpenXeroth is community-powered. Whether you're an engineer, conservationist, or volunteer, you can contribute to Hosana. Improve the pipeline, validate re-IDs, or help design integrations with platforms like Wildbook and Zooniverse.
